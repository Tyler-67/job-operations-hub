# Environment Variables

Required for production. All secrets live in Lovable Cloud / Supabase Edge Function Secrets — never commit them.

## Edge function runtime (set as Lovable Cloud Secrets)

| Name                       | Used in                                             | Notes |
|---------------------------|------------------------------------------------------|-------|
| `SUPABASE_URL`            | every function                                       | auto |
| `SUPABASE_SERVICE_ROLE_KEY` | every function                                     | auto |
| `UPTIQ_API_TOKEN`         | `_shared/uptiq.ts`                                  | Uptiq private integration token. Scopes: `users.readonly`, `contacts.write`, `conversations.write`, `calendars.write`. |
| `ACTION_TOKEN_SECRET`     | `action-token-consume`, link generation              | HMAC pepper for tap-link tokens. |
| `APP_SESSION_SECRET`      | `iframe-session`, `me`, `search`                     | HMAC pepper for app session JWT-likes. |
| `BOOTSTRAP_ADMIN_EMAIL`   | `iframe-session`                                     | Email of the first owner_admin. |
| `CRON_SECRET`             | all `cron-*` functions                               | Required header `x-cron-secret`. |
| `ALLOWED_FRAME_ANCESTORS` | `_shared/util.ts`                                   | Default `https://apps.uptiq.net`. |
| `APP_BASE_URL`            | tap-link generation (Phase 2)                       | e.g. `https://acme.uptiq.app`. |

## Frontend (`.env`, auto-generated by Lovable Cloud)

| Name                            | Purpose |
|---------------------------------|---------|
| `VITE_SUPABASE_URL`             | Edge function calls |
| `VITE_SUPABASE_PUBLISHABLE_KEY` | Anon key for client + edge invocation |
| `VITE_SUPABASE_PROJECT_ID`      | Convenience for diagnostics |

## Per-company deployment

This app is **single-tenant per company**. To onboard a new company:
1. Fork the GitHub repo (or clone the Lovable project)
2. Provision a fresh Lovable Cloud project
3. Run all migrations in `supabase/migrations/`
4. Update the seeded `locations.uptiq_location_id` to match the company's Uptiq subaccount
5. Set the secrets in the table above
6. Add the published URL as a Custom Menu Link in Uptiq with iframe params `location_id`, `user_email`, `user_name`, `phone`
